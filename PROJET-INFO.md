## 🚀 Prochaines Étapes

### 1. Installation (5 minutes)

```bash
cd portfolio-amelie
npm install
npm run dev
```

### 2. Personnalisation (30-60 minutes)

#### Informations personnelles
- [ ] Email et téléphone dans `app/contact/page.tsx`
- [ ] Liens GitHub/LinkedIn dans `components/sections/Footer.tsx`
- [ ] Liens sociaux dans `components/sections/Navbar.tsx`
- [ ] Metadata dans `app/layout.tsx`

#### Contenu
- [ ] Vos vrais projets dans `app/projects/page.tsx`
- [ ] Votre parcours dans `app/journey/page.tsx`
- [ ] Vos compétences dans `app/skills/page.tsx`
- [ ] Votre présentation dans `app/about/page.tsx`

#### Design (optionnel)
- [ ] Ajuster les couleurs dans `tailwind.config.ts`
- [ ] Changer les polices dans `app/layout.tsx`

### 3. Déploiement Vercel (5 minutes)

```bash
# Push sur GitHub
git init
git add .
git commit -m "Portfolio initial"
git push

# Sur vercel.com : Import repository → Deploy
```
